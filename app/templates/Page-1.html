Project Summary

The AI-Joke Generator is a GTP-Neo based Model that uses Natural Language Processing to create family-friendly jokes. The model takes a prompt stated by the user and uses it's advanced training to produce a result that is guaranteed to make you laugh. Using thousands of jokes sourced by Reddit's Cleanjokes subreddit, we were able to fabricate the best of the best model for our project.

Collecting The Data

We first collected from a database of clean jokes that we found online. The database later proved to be too small, so we used the reddit api, through the PRAW wrapper, to scrape our own jokes off of reddit. This doubled the size of our input, yielding funnier outputs. 

Training The Model

After collecting the data, we cleaned it, removing all unnecessary punctuation and emojis through regex and the emoji python library. We then trained the GPT-Neo model using the aitextgen library in python, with our cleaned data.
